Exercise 1.5.1

  1. For each of the following, find the free variables, if any, and decide if the given formula is a sentence. The language includes a binary function symbol +, a binary relation symbol <, and constant symbols 0 and 2.

(a) (∀x)(∀y)(x + y = 2) (b) (x + y < x) ∨ (∀z)(z < 0) (c) ((∀y)(y < x)) ∨ ((∀x)(x < y))

SOLUTION:

(a) It is a sentence, no free variables

(b) x and y are not free, therefore not a sentence

(c) x is not free on the left or, y is not free on right of or. Not a sentence

Exercise 1.5.3

  1. In mathematics, we often see statements such as sin 2 x + cos 2 x = 1. Notice that this is not a sentence, as the variable x is free. But we all agree that this statement is true, given the usual interpretations of the symbols. How can we square this with the claim that sentences are the formulas that can be either true or false?

SOLUTION:

That statement implies that we mean for all x

\[(\forall x)(sin^2x + cos^2 x = 1)\]
